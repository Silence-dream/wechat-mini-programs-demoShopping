

微信小程序黑马商城demo

## 项目简单介绍

-   使用TypeScript
-   使用eslint

## 开发
- 因为安装了eslint和prettier进行代码格式化ts文件忽略了一些文件详情请看[eslintignore](.eslintignore)
- 编写代码的使用请启动`npm run tsc`进行编码会自动的把ts代码转换成js代码
- 代码修复请使用 `npm run lint`每次提交代码之前必须进行格式校对
